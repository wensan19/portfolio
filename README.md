# Wen San Portfolio Website

This project is a personal portfolio website built using plain HTML, CSS, and JavaScript. It presents my academic work, projects, leadership experiences, community involvement, work experience, certificates, and personal interests through two viewing modes: an interactive airport check-in experience and a simpler portfolio layout.

The website was designed to reflect both creativity and personality while still presenting my work in a structured and professional way.

---

## Features

- Interactive Airport Check-In Homepage
  - Drag portfolio items into a scanner to unlock sections
  - Progress is saved using localStorage
  - Final boarding pass celebration appears after all items are checked in

- Simple Portfolio Mode
  - Cleaner and more direct browsing experience
  - Easier to navigate on smaller screens

- Portfolio Section Pages
  - About Me
  - Projects
  - Work Experience
  - Certificates
  - Community & Enrichment
  - Leadership
  - Interests
  - Recommendation Letter

- Project Showcases
  - Embedded visuals and previews
  - AI/ML, PDAS, and web development project materials
  - Notebook, PDF, and image-based previews

- Media Support
  - Images
  - Videos
  - PDF documents
  - Recommendation letter
  - Resume document

- Theme and UI Features
  - Light / dark mode toggle
  - Carousel-style content sections
  - View-all gallery layouts
  - Zoomable experience images
  - Responsive visual styling
